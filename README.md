# ESLint config

## Setup

### Install

Install the package.

```sh
npm install -D eslint eslint-config-tsuyoshi84
```

Add the package to `extends` in the ESLint config

```js
{
  "extends": ["tsuyoshi84"]
}
```

For TypeScript, use `tsuyoshi84/typescript`.

```js
{
  "extends": ["tsuyoshi84/typescript"]
}
```
